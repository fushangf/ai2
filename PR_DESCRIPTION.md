# PR 标题

feat: 提升 AI 语音绘图比赛稳定性与指令准确性

## 本日进展

- 新增语音多候选打分、识别置信度展示、噪声词过滤和重复片段去重；
- 为 3 秒静默机制增加版本锁，解决旧计时器竞态提交；
- 新增“最左、最右、第二个、最大、最小、所有”等同类对象消歧；
- 新增已验证 DrawingPlan TTL/LRU 缓存；
- AI 请求改为连接池复用，并对 429/5xx/网络错误进行有限退避重试；
- 新增客户端 DSL 白名单、55 秒看门狗及异常统一回滚；
- 新增按用户隔离的本机现场自动恢复；
- 新增 `/api/preflight` 和“系统自检”语音命令；
- 管理后台增加缓存路由统计，并修复管理员数量卡片缺失问题；
- 更新设计文档、评分点映射、演示脚本和稳定性说明；
- 自动化测试 21 项通过，离线路由基准 19/19 通过。
