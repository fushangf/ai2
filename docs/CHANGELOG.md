# Changelog

## 4.1.0 - Reliability & Accuracy Upgrade

### Added

- 多候选语音识别打分、置信度展示与低信息量噪声过滤；
- 三秒静默版本锁，避免旧计时器竞态提交；
- “最左、最右、第二个、最大、最小、所有”对象消歧；
- 已验证 DrawingPlan TTL/LRU 缓存；
- HTTP 连接池、429/5xx 退避重试；
- 55 秒客户端请求看门狗和所有异常事务回滚；
- 按用户隔离的本机现场自动恢复；
- `/api/preflight` 与“系统自检”语音命令；
- 管理后台缓存路由统计；
- 新增缓存、空间消歧与自检测试。

### Fixed

- 修复管理员后台缺失 `adminUsers` 元素导致脚本报错的问题；
- 修复执行异常时可能留下部分图元的问题；
- 修复不同用户共用一个本机恢复键的隐私风险。

### Verified

- 自动化测试 21 项通过；
- 离线路由基准 19/19 通过。
