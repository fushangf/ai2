# 安全说明

## 1. 密钥与密码

公开仓库禁止提交：

- `.env`；
- 七牛云或其他 AI API Key；
- MySQL 真实用户名和密码；
- 管理员真实密码；
- 生产环境 `AUTH_SECRET_KEY`；
- 包含敏感配置的截图、日志或测试数据。

仓库只保留 `.env.example` 和 `.env.competition.example` 占位模板。

若密钥曾进入聊天、Git 历史、公开附件或压缩包，应视为已经泄露并立即重置。仅从最新提交删除文件并不能消除历史泄露。

## 2. 评委演示模式

`COMPETITION_KIOSK_MODE=true` 只用于本机比赛演示：

- 演示登录接口默认只允许 `127.0.0.1`、`::1`、`localhost`；
- 不使用固定演示密码；
- 公开部署必须关闭；
- 不应通过反向代理把演示接口暴露到公网。

## 3. AI 输出安全

- 模型只输出 DrawingPlan JSON；
- 不执行 `eval()`；
- 不接收脚本、HTML、SVG 字符串和网络地址；
- Pydantic 校验结构；
- Guardrails 限制颜色、坐标、尺寸、ID、位移、缩放和操作数；
- 用户语音放在 `<voice>` 标签中并作为数据处理，降低提示注入影响。

## 4. 认证与限流

- 密码使用 PBKDF2 哈希；
- token 带过期时间；
- 每次请求重新读取用户状态，被封禁用户现有 token 也会失效；
- 绘图接口按用户进行滑动窗口限流；
- 管理员不能封禁管理员账号。

## 5. 生产部署

- 使用 HTTPS；
- 设置独立数据库账号和最小权限；
- 使用随机、至少 32 字符的 `AUTH_SECRET_KEY`；
- 修改管理员初始密码；
- 限制 CORS 域名；
- 使用服务器密钥管理系统注入环境变量；
- 定期备份数据库；
- 不在健康接口返回数据库连接串或密钥。
