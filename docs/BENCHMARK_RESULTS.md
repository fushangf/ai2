# 离线命令路由基准

运行环境：本次构建容器，Python 3.13。命令：

```bash
python benchmarks/evaluate_command_router.py
```

本次运行结果：

- 样例数：19；
- 本地路由 / AI 回退分类与操作拆解：19/19；
- 中位路由耗时：约 0.102 ms；
- P95 路由耗时：约 0.216 ms。

覆盖样例包括：

- 向四个方向移动与指定像素移动；
- 放大、缩小与指定百分比；
- 顺时针旋转；
- 中文颜色变更；
- 删除与清空；
- 一句话三步骤编辑；
- “最右边的云朵”“第二个云朵”“所有云朵”等同类对象消歧；
- 复杂风格、新场景和海报类请求正确回退 AI。

> 限定说明：该基准只测试文本进入本地路由后的分类和 DSL 拆解，不包括麦克风、浏览器语音识别、网络 AI 延迟或 Canvas 渲染，因此不能作为真实语音识别准确率或完整端到端延迟数据。
