# AI 语音绘图工具 4.2 更新说明

本次更新完成两项需求：

1. 新增 **语音交流模型 V2**
   - 语音命令：`进入交流模式`、`退出交流模式`、`清空交流记录`
   - 继续沿用“开始 -> 自然说话 -> 静默 3 秒自动提交”的纯语音闭环
   - 交流模式下不修改画布，只返回自然语言语音回复
   - 新增后端接口：`POST /api/voice-chat-v2`
   - 新增管理员统计：`voice_chat_route_count`，最近调用记录显示 `语音V2`

2. MySQL 宿主机端口改为 **3307**
   - `docker-compose.yml` 改为 `${MYSQL_HOST_PORT:-3307}:3306`
   - `.env.example`、`.env.competition.example`、自用版 `.env.personal.example` 新增 `MYSQL_HOST_PORT=3307`
   - 宿主机访问 MySQL 时使用 `127.0.0.1:3307`

## 语音交流 V2 使用方式

- 说：`进入交流模式`
- 再说：`开始，帮我总结这幅画的亮点`
- 静默 3 秒后自动回复
- 说：`退出交流模式` 返回绘图模式

## 验证结果

- Python 自动化测试：`23 passed`
- 前端脚本语法检查：通过

## 兼容性说明

- 原有绘图模式、登录注册、管理员后台、作品管理、系统自检保持可用
- 绘图模型与语音交流 V2 模型默认共用同一个 AI Provider，可通过 `VOICE_CHAT_MODEL_V2` 单独覆盖
